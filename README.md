# Empresa app
### Aplicativo teste para busca na web e exibição de dados na web, foram utilizadas as bibliotecas Retrofit para requisiçes e Glide para renderizaço de imagens. Foi utilizada a interface de searchview disponibilizada pelo Material Design da Google para uma melhor UX nas buscas

## Melhorias a serem feitas
### como melhoria eu utilizaria programação reativa(AndroidRx) para as requisições
